# Mindusyry Logic Script Wiki  
## 介绍  
这是一个编译器  
用于把一种自制的脚本语言转化为如果Mindustry Logic** _(即像素工厂逻辑)_  
> [!NOTE]目前处于开发中  

## 如何使用  
首先你要从github上下载编译器  
在仓库的[`rlease页面`](https://github.com/zmzm-zm/MindustryLogicScript/releases)进行下载  
如果没有，那就是我还没做出demo  
你需要编写一个以`.mls`结尾的文件  
语法可见[语法章节](#语法)
